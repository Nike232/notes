## CSS基础知识1

* 层叠样式表（css）是一种样式表语言，用来美化内容

* 书写位置（基础）:

  * title 标签下面(head里面)添加style双标签，style 标签里面写css代码

  * ~~~
    <style>
    /*选择器{}*/
    p{
    	/*css属性*/
    	color:red;
    }
    </style>
    ~~~

  * 属性名和属性值同时出现叫做键值对

  * color 文字颜色

  * font-size 字号

* CSS的引入方式:
  * 内部样式表：写在style标签里面(学习的时候使用)
  * 外部样式表：==开发使用==
    * css代码写在单独的css文件里面
    * 在html里面用link标签引入
      *  `<link rel="stylesheet" href="./my.css">`
      * link标签放的位置跟内部引用放的位置一样
  * 行内样式:配合js使用
    * css 写在标签的style属性里面

* ==css选择器==
  * 作用：查找标签，设置样式
  * 标签选择器
  * 类选择器
  * id选择器
  * 通配符选择器

* 标签选择器
  * 使用标签名作为选择器 ->选中**同名标签**设置**相同的样式**
  * 会选中所有同类的标签类型，设置成相同的样式（不能差异化）

* 类选择器
  * 可以**差异化**设置标签的显示效果
  * 定义类选择器:  .类名
  * 使用类选择器: 标签添加 class="类名"
  * **注意一个标签只能有一个class 但是可以有多个属性值 用空格隔开**
  * ==开发习惯:类名见名知义,多个单词可以用-来连接

* id选择器
  * 实际开发过程中,一般用类选择器设置css样式
  * 而id选择器一般配合**js**使用，很少用来设置css样式
  * 定义id选择器：#id名
  * 使用id选择器: id="id名"
  * **同一个id选择器在一个页面中只能使用一次**

* 通配符选择器
  * 作用：**查找页面所有标签，设置相同的样式**
  * 使用：*
  * 不需要调用，浏览器自动查找页面所有标签设置成相同的样式
  * **不会改变单独设置了的样式，只是改变没有设置的样式**
  * **注意如果是有一定设置的样式，但是没有设置到的样式也是通配符设置的样式**
  * 可以理解为改变了默认样式
  * 一般在开发初期，清除默认样式时使用

* 几个新属性：
  * width
  * height
  * background-color 背景色

* **文字控制属性**
  * font-size 文字大小   最常用单位px(==必须要有单位==) 谷歌浏览器文字有默认大小(16px)
  * font-weight 文字粗细 属性值可以是数字也可以是关键字
  * 数字（开发使用）400正常 700加粗 关键字 normal bold
  * font-style 字体倾斜 normal 不倾斜 ltalic 倾斜（一般用不上 因为不太好看）
  * line-height 行高 设置多行文本的间距 属性值：1.数字+px 2.数字(当前font-size属性值的倍数)
  * 行高由上间距、**文本高度**、下间距组成 行高的测量方法：从一行文字的最低端量到下一行文字的最顶端
  * 使用行高实现垂直居中:**行高属性值等于盒子高度属性值**（根据原理如果换行了就不能垂直居中了）(所以这个技巧只能用于单行文字垂直居中)
  * font-family 字体族 属性值就是字体名 (拓展 了解):属性值可以书写多个字体名，用逗号隔开，执行顺序是从左往右查找
    * 在属性最后设置一个字体族名，网页开发建议使用==无衬线字体==（sans-serif）开发时直接复制
  * font 字体复合属性  使用场景:**设置网页文字公共样式**，哪里不一样再修改哪里的属性就可以了
    * 就是一种简写的方式，一个属性对应多个值的写法，中间用空格隔开
    * font:是否倾斜 是否加粗 字号/行高 字体（==必须按顺序书写==）(注意这==/==不是或就是写法)
    * **注意字号和字体必须书写，否则font属性不生效**
  * text-indent 文本缩进 数字+px 或者 ==数字+em==(推荐 1em=当前标签的字号大小)
  * text-align 文本对齐
    * left 左对齐(默认)
    * center 居中对齐
    * right 右对齐
    * **本质是控制内容的对齐方式，属性要设置给内容的父级**
  * text-decoration修饰线
    * **none** 无 可以删除a默认的下划线
    * **underline** 下划线
    * line-through 删除线
    * overline 上划线
  * color 颜色
    * 颜色关键字 英文单词 red、green 学习测试用
    * rgb表示法 rgb(r,g,b) 表示三原色 取值0-255 了解
    * rgba表示法 **rgba(r,g,b,a)** a表示透明度，取值0-1(小数)  开发使用，实现**透明色**
    * 十六进制表示法 **#RRGGBB** 开发使用（从设计稿复制）两个数一组，如果两个数是一样的可以简写成一个数
