# 位运算

### 一、获取第i位的数字（特殊：符号位）

1. ==(a>>i)&1或者a&(1<<i)==;最后得到的结果为1或者0

2. 特殊地获取符号位并拓展,==x>>31；==
3. !!(x&(1<<31));用于判断x是否为负数其中 1<<31为符号位掩码

### 二、设置第i位为1

**强制赋值用|运算**

==a=a|(1<<i);==

### 三、设置第i位为0

将设置第i位为1的操作进行取反即可

==a=a&(~(1<<i));==

### 四、将第i位取反

==a=a^(1<<i);==

任何一位与0进行异或操作结果不变，与1进行异或操作结果取反
